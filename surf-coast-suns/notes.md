# Surf Coast Suns Cricket Club — Project Notes

- **Project:** Surf Coast Suns Cricket Club Website
- **Goal:** Public-facing website for the Surf Coast Cricket Club (North Torquay). Covers club info, teams, fixtures, and contact.
- **Stack:** Single HTML file, vanilla CSS, vanilla JS
- **Design direction:** Light editorial aesthetic. Warm orange/red accent colours. Bebas Neue for display headings, Inter for body text. Card-based layout.
- **Target audience:** Club members, local community, prospective players.
- **Status:** Active — index.html built and iterated on. Assets folder contains hero image, action photo, and club logo.

## Teams covered
- Senior teams (men's)
- U11 juniors
- U17 juniors

## Key decisions
- Women's section and Official Club Website footer link have been removed.
- About section feature chip tiles removed for cleaner layout.
- Contrast issues fixed following UX audit.
